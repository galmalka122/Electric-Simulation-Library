# Electric Simulation Library

This C++ library provides tools for simulating electrical circuits. It includes a flexible component model and a circuit simulation framework.

## Features

- Abstract component model supporting multi-terminal components
- Circuit class for managing connections between components
- CMake-based build system for easy integration

## Getting Started

### Prerequisites

- C++17 compatible compiler
- CMake 3.10 or higher

### Building the Library

```bash
mkdir build && cd build
cmake ..
cmake --build .
```
